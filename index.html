What This App Does

Well-Formed Scale Explorer builds and plays well-formed scales from either generator data or scale-step data.

The scale updates in the background as values are applied, making it easy to modify, compare, and reshape scales.

Build Method

Generator Build

Specify the scale by period p and generator g in frequency ratios, or by p and logp(g).

Choose a compatible cardinality N from the available values.

Scale Step Build

Specify the scale by:

  • period
  • cardinality
  • number of A steps
  • either the relative proportion A:B or the size of A

Applied changes update the scale in the background, so the two methods can be used flexibly as complementary ways of shaping the same object.

Base Frequency

The base frequency sets the sounding register of the scale.

You may enter:

  • a note name in scientific pitch notation, such as F3, C4, Bb4
  • or a frequency in Hz

Information Panels

Cardinalities

  • N: total number of scale tones in the period
  • #A, #B: numbers of occurrences of the step types
  • A/B: logarithmic ratio of the two step sizes

Values

Shows p, g, A, and B as frequency ratios and cents values.

Keyboard

The keyboard allows a range of two full periods for demonstration. The keyboard displays:

  • white keys for scale tones
  • black keys for auxiliary tones between adjacent scale tones

The step pattern is shown above the keyboard.

Click a key to play it.

If Computer keyboard is enabled, click once in the keyboard area before using the computer keyboard.

Mode

The mode control selects the tonic within the scale.

Mode order steps through modes either:

  • in scale-step order
  • in generator order

Keyboard Labels

  • note numbers
  • Hz
  • Hz/Base
  • cents

Playback

  • up
  • down
  • up/down
  • generator order

Duration controls playback note length.

Timbre selects the instrument sound.

Computer Keyboard

If enabled:

  • lower-case keys play white keys
  • shifted keys play the black key to the right

This mapping is limited by the number of available physical keys, so very large scales may extend beyond the computer-key range even though they still appear on screen.

Special Case: Degenerate Well-Formed Scales

Every equal tempered scale is understood to be a degenerate well-formed scale. Whenever logp(g) is rational, such a scale will be the largest cardinality. Such scales do not transport back predictably from Scale Step Build to Generator Build.
